# push/pull/pull request
### push:
- یعنی ارسال تغییراتی که روی سیستم محلی فرد انجام شده به مخزن گیتهاب
### pull:
- یعنی دریافت اخرین تغییرات از مخزن گیتهاب و ادغام کردن ان با نسخه ی محلی فرد
### pull request:
- یعنی درخواست برای ادغام تغییراتی که روی شاخه یا فورک ایجاد شده به پروژه ی اصلی
## تفاوت ها:
- در پوش شخص تغییراتش را ارسال میکند
- در پول تغییرات دیگران دریافت میشود
- در پول ریکوست درخواست بررسی تغییرات و اضافه شدن ان به پروژه ی اصلی داده میشود
